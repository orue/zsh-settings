## My ZSH Configuration

In your home directory, create a `.zshenv` file with the following content:

```sh
ZDOTDIR=$HOME/.config/zsh
```

Then, clone this repository into the `.config/zsh` directory:

```sh
.config/zsh
├── .gitignore
├── .zshrc
├── aliases.zsh
├── aws.zsh
├── exports.zsh
├── functions.zsh
├── git-aliases.zsh
├── nvm.zsh
├── python-venv.zsh
├── README.md
├── starship.toml
├── transient-prompt.zsh
├── user-aliases.zsh
└── variables.zsh
```
